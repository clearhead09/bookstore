# Second-Hand Bookstore

A mobile-first inventory and point-of-sale web application for second-hand bookstores.

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- Git

## Current Features

- 📷 Barcode scanning using a phone camera
- 📖 Book lookup by ISBN
- 📱 Mobile-friendly interface
- 🖼 Book cover display

## Planned Features

- Add new books to inventory
- Edit book details
- Inventory management
- Sales tracking
- User login

## Running the Project

1. Create a virtual environment
2. Install dependencies
3. Run:

```bash
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
